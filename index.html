HTML Reference

HTML References

Tag Name Description
<em> Emphasis Italicizes the words within.
<div> Div Element Divides an HTML document into sections, can contain other content
<h1-h6> Headings 1-6 Six different levels of heading that HTML can use.
<p> Paragraph Element An element that contains and displays a block of text.
<a> Anchor Element Used to create hyperlinks to other webpages, files, or any other URL.
<img> Image Element Embeds images in documents. It is an empty element meaning it should not have a closing tag

HTML Attributes

Tag Name Description
<ul> Unordered List Used to turn an ordered list (numbered) into a bullet list.
<target> Target Attribute Specifies where a hyperlink should be opened. "_blank" will tell the browser to open it in a new tab, or a new window if the browser is old.
<src> Source Attribute Dictates where the element above should source the relevant content, i.e a video or img element.

Table References

Tag Name Description
<table> Table Wrapper element for HTML tables
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table data container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Tag Name Description
<colspan> Column Span Defines how many columns a td element should span.
<rowspan> Row Span Defines how many rows a td element should span.
